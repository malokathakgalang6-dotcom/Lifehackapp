# 📱 Life Hack or Urban Myth App

## 📌 Purpose
This Android app helps users identify whether common life hacks are true or false using an interactive quiz.

---

## 🎮 Features
- Flashcard-style quiz
- True/False answer buttons
- Instant feedback after each answer
- Score tracking system
- Final score screen
- Review screen with explanations

---

## 🧠 How it works
1. User opens the app
2. Clicks Start Quiz
3. Answers True or False questions
4. Receives feedback immediately
5. Moves through questions using Next button
6. Final score is displayed
7. User can review answers

---

## 🛠 Technologies Used
- Kotlin
- Android Studio
- XML Layouts
- GitHub

---

## 🧪 Testing
The app was tested using an Android emulator to ensure:
- Buttons work correctly
- Navigation flows properly
- Score is calculated accurately
