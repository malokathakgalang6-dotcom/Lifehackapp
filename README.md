# Life Hack or Urban Myth App

## 📱 Purpose
This app helps users identify whether a life hack is real or a myth using a quiz format.

## 🎮 Features
- Flashcard quiz
- True/False answers
- Score tracking
- Feedback messages
- Review screen with explanations

## 🧠 How it works
Users answer questions and receive feedback. At the end, a score is displayed with a message.

## 🛠 Technologies
- Kotlin
- Android Studio
- GitHub

## 🔄 GitHub Actions
Automated builds are used to ensure the app runs correctly.
